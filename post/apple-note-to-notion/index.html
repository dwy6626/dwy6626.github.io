匯出成 .md 檔

首先我們需要一個小幫手：

  • Notes Exporter

這個 app 可以一鍵幫你匯出備忘錄裡面所有的筆記，UI 美美的，操作起來也方便，按下去之後等就好~~，真的很有中國人的風格 XD~~

原先是在 Bear.app 的網站看到的，不過既然都是 Markdown，那我就借來用了。

匯出之後

但總有一些不順利的，例如說匯出之後格式其實還是會跑版，但這時候已經是文字檔了，身為一個 Programmer 就算不是大神，但是寫寫腳本應該還是會的：

  • dwy6626/Markdown-Format-Fixer

用法很簡單：

python main.py [your folder]

當然前提是你要裝好 Python，好像還有你要會用 Terminal，一般來說 macOS 都有內建，但我用的是 Python 3.7，注意一下版本，如果是 2.X 的話我不保證能跑。

順利的話，你會拿到一堆 .md 檔。
