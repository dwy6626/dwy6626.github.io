這裡搜集了一些我會用到的 Notion 的小技巧。

換行

跟 FB 對話框一樣，如果你想要換行，但不想產生新的 Block，
就按 shift + enter，就會在同一個 Block 內換行，可以發現這行和上行距離比較小。

而且不用像原生 Markdown 一樣要硬要加兩個空白在後面（十分違反直覺），很方便。

Link a page 和 mention a page 的差別

Linked page 會被視為 subpage，會出現在 sidebar，所以一個 page 內不能多次重複 link 同一個 page。

但是 mention a page 不會將其視為 subpage，所以可以一直提到那個 page。

因為很重要所以可以 mention 很多次。

A linked page will be shown as a subpage, a mentioned page will not. For this reason, you can @mention the same page multiple times on a single page, but you can only include one linked page. Does that make sense?

— Notion (@NotionHQ) March 7, 2019

全域搜尋

Quick Find 在頁面右上角，很遠對吧，又不能用 cmd + f（會叫出文章內搜尋），沒關係，可以用 cmd + p 來快速使用全域搜尋（老實說有點難記 XD）
（因為我的電腦是 mac，在 windows 系統的話，應該是 ctrl + p）

編輯頁面的熱鍵組合

寫上幾個比較實用的，剩下放在連結裡：

  • cmd + option + 1~8 可以轉換 Block 格式（ 9 會變成新的頁面）
  • 按著 option 拖拉 Block 可以直接拉出一個新的內容一樣的 Block
  • cmd + a 可以全選整個 Block 內容，如果按第二次，會全選整個頁面
  • 和其他文字編輯器一樣，可以用 option 或 cmd + → 或 ← 快速移動游標
  • Esc 可以快速選取整個 Block，如果要刪除就在選取 Block 的時候按 Del 或 Backspace 就好了，就不需要用到滑鼠

（因為我的電腦是 mac，在 windows 系統的話，應該把 cmd 換成 ctrl 就可以了）
